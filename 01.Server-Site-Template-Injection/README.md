# ¿Cómo levantar el Lab?

## Darle permisos de ejecución a los dos script
* flask_form
```
chmod +x ./flask_form/run_flask.sh
```

* flask_form_vuln
```
chmod +x ./flask_form_vuln/run_flask.sh
```

## Ejecutar ambos archivos
* flask_form
```
./flask_form/run_flask.sh
```

* flask_form_vuln
```
./flask_form_vuln/run_flask.sh
```
